Nav Generator

How it works

  1. Enter the number of level-1 categories you will need and then click the "Generate" button
  2. Once the categories are generated enter the name of the level-1 category and the number of level-2 categories that you will need. Clicking the "Generate level-2" button will create the respective amount of level-2 categories that you want
  3. Fill out all of the information for level-2 categories. Once you are finished with that - click the "Parse HTML" button which will output the raw HTML code that you need
